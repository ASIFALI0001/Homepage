About

This is a smart attendance system designed for modern classrooms. It uses Face ID recognition with Raspberry Pi to make marking attendance fast, secure, and contactless.

Students and teachers each get their own dashboards:

  • Students can view their subject-wise attendance anytime.
  • Teachers can manage class records, verify issues, and track attendance effortlessly.

If Face ID ever fails, students can report an issue directly through the system for manual approval. It's simple, reliable, and built for the future of education.
